# Atlasとは (Introduction)

Atlasは、各種プロジェクトの公式マニュアルや技術ドキュメントを素早く構築・公開するための**ドキュメント基盤（金型）** です。

## 主な特徴

- **Docs as Code**: [mdbook](https://rust-lang.github.io/mdBook/) を採用しており、普段書き慣れた Markdown 形式でドキュメントを記述できます。
- **共有テーマ (`atlas-common`)**: 全プロジェクトで統一された美しいUI/UXを提供するための共通アセット（テーマ設定など）をサブモジュールとして統合しています。
- **無停止自動デプロイ**: GitHub Actionsを利用し、`main` ブランチにプッシュするだけでVPSへ自動的かつ無停止でデプロイされる仕組み（コンベア）が組み込まれています。

このリポジトリ自体をコピー（複製）することで、誰でも簡単に新しいプロジェクトのドキュメントサイトを立ち上げることができます。
